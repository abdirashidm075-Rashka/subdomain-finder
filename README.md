# Infrastructure Intelligence: Automated Subdomain Recon Fuzzer

## 📝 Description
This passive reconnaissance utility maps organizational attack surfaces. By processing a target root domain alongside an active dictionary stream, the script handles localized name resolution requests to discover unadvertised subdomains, staging environments, or shadow IT infrastructure assets.

## 🛠️ Features
* **DNS Dictionary Parsing:** Automates brute-force string creation across high-frequency host configurations.
* **Low-Overhead Host Interrogation:** Leverages standard host lookup hooks to probe remote names without generating noisy application server logs.
* **Network Inventory Generation:** Outputs live host mappings to accelerate perimeter assessment and reporting.
